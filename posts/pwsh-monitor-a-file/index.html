[pwsh] Monitor the latest changes of a file (tail equivalent)

Feb 19, 2021 - pwsh powershell snippet 
1Get-Content -Path C:\Windows\System32\dhcp\DhcpSrvLog-fri.log -Tail 20 -Wait

-tail will display the latest 20 lines of the file and -wait to see updates in real time.
